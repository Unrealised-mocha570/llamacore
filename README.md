# 🤖 llamacore - Run local AI models with ease

[![Download llamacore](https://img.shields.io/badge/Download-Llamacore-blue.svg)](https://github.com/Unrealised-mocha570/llamacore)

Llamacore provides a simple interface to run AI models on your own computer. You do not need to upload data to the cloud or pay for subscriptions to use advanced artificial intelligence. The software organizes your models into a clear list, manages background tasks, and allows you to build sequences of commands for complex projects.

## 🛠 Features

Llamacore simplifies the process of interacting with locally stored models.

- Chat interface: Talk with your models using a clean, familiar window.
- Model Manager: Store your GGUF files and switch between them with one click.
- Server Control: Start or stop the underlying model engine without using a terminal window.
- File Conversion: Change models from Hugging Face into the specific format needed for local play.
- Performance Monitor: Watch how much memory and processor power your current tasks use.
- Workflow Builder: Chain simple actions together to automate multi-step artificial intelligence tasks.

## 🖥 System Requirements

Your computer requires specific hardware to run artificial intelligence models well.

- Operating System: Windows 10 or Windows 11.
- Processor: A modern dual-core chip or better.
- Memory: At least 8 gigabytes of system RAM.
- Storage: 500 megabytes for the application, plus extra space for your AI model files.

## 💬 Using the Chat feature

Once you open the software, go to the chat screen. The top bar shows your current active model. If no model appears, select one from your library. The program will load the model into your system memory. Once the status indicator turns green, you can type your questions into the text box at the bottom. Press the Enter key to send the message. The model will respond based on the data it contains.

## ⚙️ Managing models

The model tab contains your file library. To add a model, move a GGUF file into the folder designated in your settings menu. The software scans this folder and displays all available models in the list. You can rename your models, remove them, or view specific details about their memory size and creator.

## 📂 Converting models

Not all AI models work with local software immediately. Many models exist in formats that require changes before use. Use the conversion tool provided in the menu to select a downloaded model file. The application converts the file into the correct format automatically. This process takes a few minutes depending on the size of the model file and the speed of your computer processor.

## 📈 Monitoring your hardware

Local AI creates high demand on your computer hardware. Click the monitor button to view your current usage. You will see how much memory the model consumes and how much work your processor does per second. If your interface slows down, close other programs to free up memory for the model.

## 🧩 Building workflows

Workflows allow you to automate long tasks. You can define a series of steps where one model summarizes a document, another translates it, and a third creates a report based on those results. Create a new workflow, drag the action blocks into the sequence you prefer, and link them together. Save the workflow to run it again whenever you require that specific chain of tasks.

## 🛡 Security and Privacy

Your data stays on your machine at all times. Since this application runs locally, no information travels across the internet to servers owned by other companies. You control the connection. If you wish to use the software while offline, disconnect your internet and continue your work without interruption.

## 💡 Troubleshooting

If the application fails to start, verify your Windows version. Ensure you have the latest updates from Microsoft. If you encounter errors when loading a model, verify that the model file is not damaged. Try to clear the cache file located in the settings menu if the interface becomes unresponsive. Always ensure you have enough free disk space before you try to start a large model.

## ❓ Frequently Asked Questions

Do I need an internet connection?
You only need the internet to download the software and to download model files. You can turn your internet off after the models reside on your hard drive.

Where can I find more models?
Many websites host open-source models. Look for models labeled GGUF to ensure they work with this software.

Is this software free?
Yes. The project remains open for everyone to use and modify for their own needs.

Can I move the settings folder?
You can change the default storage path within the application preferences menu. Choose a location on a drive with ample available space.

Does this work on older computers?
Older computers might struggle to run large models quickly. Try smaller model files to see if your system can handle them.
